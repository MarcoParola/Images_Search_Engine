\section{Introduction}
In this project we implemented a Web Search Engine working on "ArtImages" dataset and "MirFlickr" dataset as distractor. 
The Web Search Engine is based on the Locality Sensitive Hashing (LSH) index and on the pre-trained convolutional neural network ResNet50 with weigths computed on the "Imagenet" dataset.


\section{Data Analysis and Preprocessing}
The Art dataset is composed of 5 categories of images, corresponding to the 5 classes:
\begin{itemize}
\item Engraving
\item Sculpture 
\item Iconography
\item Painting 
\item Drawings
\end{itemize}
\subsection{Data cleaning}
After a preliminary analysis of the dataset, we realized that some images are corrupted both on trainingset and validationset, so we detected and deleted them using the following script.

\section{Sequential Scannig}
The first implementation of our image search engine is based on a numpy array as data structure and a sequential scanning as search algorithm, using the following similarity distance:
\begin{itemize}
\item Euclidian distance
\item Cosine similarity
\item Hamming distance
\end{itemize}
